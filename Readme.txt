PicRes by Martial Demolins
Version: 1.1
Source + download page: https://github.com/Folcogh/PicRes
Statically built against Qt 5


This program is intended to resize multiple images at once, in an easy way:
1. drag and drop files
2. set resizing method (absolute or relative)
3. click "Resize"

Supported image formats: BMP, JPG, PNG, PPM, XBM, XPM

/!\ Warning: resized files replace original ones /!\ 

Don't hesitate to ask me for improvements.
